# Console扫雷
上班摸鱼使用浏览器F12 Console控制台进行扫雷游戏操作。  
既然有敲键盘的动作又有停留片刻的思考，让周围同事和领导感觉你在工作。

# 使用方式
## 第一种
最简单粗暴，把[index.js]('./index.js')中的代码复制粘贴到Console中就能使用了。

## 第二种方式
使用`Sources`中的 `Snippets` 添加外挂的js程序，将[index.js]('./index.js')代码复制进去之后，右键 `Run`运行即可。  
好处：配置一次可以在所有页面中使用
